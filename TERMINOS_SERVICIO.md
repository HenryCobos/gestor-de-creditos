# Términos de Servicio - Gestor de Créditos

**Última actualización:** 4 de agosto de 2025

## 1. Aceptación de los Términos

Al descargar, instalar o usar la aplicación **Gestor de Créditos** ("la Aplicación"), usted acepta estar sujeto a estos Términos de Servicio ("Términos"). Si no está de acuerdo con estos términos, no debe usar la Aplicación.

## 2. Descripción del Servicio

### 2.1 Funcionalidad
Gestor de Créditos es una aplicación móvil que permite:
- Gestionar carteras de clientes
- Registrar y dar seguimiento a préstamos
- Calcular cuotas y fechas de vencimiento
- Generar reportes financieros
- Enviar notificaciones de recordatorio

### 2.2 Disponibilidad
- La Aplicación está disponible para dispositivos iOS (iPhone)
- Requiere conexión a internet para algunas funcionalidades
- Los datos se almacenan localmente en su dispositivo

## 3. Uso Aceptable

### 3.1 Uso Permitido
Usted puede usar la Aplicación para:
- Gestionar sus propios préstamos y clientes
- Mantener registros financieros personales o de su negocio
- Generar reportes para análisis financiero

### 3.2 Uso Prohibido
No debe usar la Aplicación para:
- Actividades ilegales o fraudulentas
- Violar derechos de propiedad intelectual
- Intentar hackear o comprometer la seguridad
- Usar la Aplicación para fines comerciales sin autorización
